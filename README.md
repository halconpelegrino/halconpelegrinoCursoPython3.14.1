# Ejercicio_Curso
![CI](https://github.com/halconpelegrino/halconpelegrinoCursoPython3.14.1/actions/workflows/ci.yml/badge.svg)


Entorno virtual: `curso`

## Comandos principales
- Activar entorno: source curso/bin/activate
- Instalar dependencias: make install
- Formatear: make format
- Lint: make lint
- Ejecutar ejemplo: python main.py
- Tests: make test

## VS Code
- Abrir proyecto: code .
- Seleccionar intérprete: Ctrl+Shift+P → Python: Select Interpreter → .../curso/bin/python
